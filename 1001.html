Project Highlights

  • Integrated ultrasonic sensors for obstacle detection and path rerouting.
  • Developed a servo-controlled lifting arm optimized for bag lifting.
  • Performed stress-strain analysis to determine weight distribution and torque limits.
  • Implemented PID control for precise movement and line tracking.
  • Programmed in C++ for task execution
