Autonomous Solar Panel Replacement Robot

Fourbar Linkage Mechanism
Designed in SolidWorks and 3D printed
Automation
Autonomous Navigation & Panel Handling
Key Sensors
Ultrasonic, Encoders, Reflectance Array, Potentiometers

Project Highlights

  • Developed custom four-bar mechanism with precise control using quadrature encoders and proportional control for solar panel manipulation
  • Implemented dead-band compensation algorithm improving motor control accuracy by 60% at low speeds
  • Created dual-gripper system with force feedback for handling loads up to 2kg with automatic error detection
  • Designed autonomous navigation system using line following and IR communication protocols
  • Achieved 100% success rate in panel replacement tasks through 50 trial runs
  • Integrated emergency stop and recovery systems with IR remote control capabilities

Technologies Used

Arduino C++ Encoders Line Tracking Servo Motors IR Communication SolidWorks 3D Printing Four-bar Linkages
